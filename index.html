About Rythu Bandam

Rythu Bandam is a digital initiative created as part of a web development internship project. The purpose of this platform is to assist farmers by offering simplified access to resources like:

  • Daily weather forecast integration
  • Live crop price tracking
  • Direct access to government agricultural welfare schemes
  • Mapping of nearby agricultural support shops

This project is built using HTML, CSS, and JavaScript as part of the ApexPlanet 45-day internship program.
